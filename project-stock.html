About the Project

Stock market prediction is notoriously difficult due to volatility. This project utilizes Long Short-Term Memory (**LSTM**) networks, a type of recurrent neural network capable of learning order dependence in sequence prediction problems.

The model analyzes historical price data, volume, and moving averages to forecast future trends, providing traders with an additional tool for technical analysis.
